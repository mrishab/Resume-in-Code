<h3>
Caesar Cipher Breaker Tool (C++)
</h3>
<small>
Introduction to Programming - December 2016
</small>
<p>
It is Brute-force tool built in C+. It can find the shift value for almost any Caesar cipher that contains a two-letter word. The tool brute forces the two-letter word to match it with anyone in the predetermined list which is stored as array. Once, a match is found, it will display the shift value.

Link
</a>
